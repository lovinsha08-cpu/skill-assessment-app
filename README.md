# EV Skill Assessment App 🚀

An intelligent skill assessment platform that helps users discover their technical strengths, evaluate their current skill level, and identify areas for improvement through AI-powered analysis.

## 🌐 Live Demo

🔗 https://ev-skill-assessment-app.streamlit.app/

## 💡 Why this project?

Many students and professionals learn multiple technologies but struggle to understand:
- What skills they currently possess
- Their actual proficiency level
- Which areas need improvement
- What they should learn next

This project aims to bridge that gap by analyzing resumes, extracting relevant skills, assessing knowledge through quizzes, and providing personalized insights.

## ✨ What it does

### 📄 Resume Intelligence
Users can upload their resume, and the application extracts relevant technical skills automatically.

### 🧠 Skill Analysis
The extracted skills are analyzed and matched with a skill reference database to understand the user's technical profile.

### 📝 Personalized Assessment
Based on identified skills, the application generates assessments to evaluate understanding and proficiency.

### 📊 Skill Dashboard
Users can view their performance, strengths, and improvement areas through an interactive dashboard.

### 📚 Learning Guidance
The platform recommends areas where users can focus to improve their technical capabilities.
